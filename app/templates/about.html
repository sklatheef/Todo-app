How This App Works

This Task Manager is a full-stack Django application designed to help you stay organized. It uses a relational database to store your tasks and a secondary history table to archive your deletions.

1. Add & View Tasks

On the Home page, you can submit a task name and description. These are saved to the Task_Db table and displayed immediately on the All Tasks list.

2. Edit & Details

You can modify any existing task. Clicking "Edit" retrieves the specific object by its ID, allows changes, and saves the updated data back to the database.

3. Smart Deletion (History)

Unlike standard apps, when you click Delete, the app doesn't just erase the data. It first creates a copy in the History_Db table before removing it from the active list. This ensures you never lose a record of what you've done!
